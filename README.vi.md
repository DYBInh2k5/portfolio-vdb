# Portfolio VDB — Hướng dẫn nhanh (Tiếng Việt)

Đây là một dự án Next.js cho portfolio cá nhân.

Chạy môi trường phát triển:

```bash
npm install
npm run dev
```

Mở http://localhost:3000 để xem trang.

Để build production:

```bash
npm run build
npm start
```

Tệp chính để chỉnh sửa: `src/app/page.tsx`.
